مدت زمان تدریس: ۱۲۰ دقیقه | هدف: اجرای اولین کد PHP، درک متغیرها، انواع داده و ساختارهای شرطی

بخش اول: زمان‌بندی جلسه (۱۲۰ دقیقه)

این جلسه اولین برخورد دانشجویان با دنیای واقعی PHP است. زمان‌بندی باید به‌گونه‌ای باشد که دانشجو همزمان با یادگیری تئوری، کدنویسی عملی را نیز تجربه کند.

  1. نصب و راه‌اندازی XAMPP (۲۰ دقیقه): دانلود، نصب، تست اجرای Apache و MySQL. ایجاد اولین فایل PHP و اجرای آن در مرورگر.
  2. آشنایی با سینتکس PHP و متغیرها (۳۰ دقیقه): تگ‌های باز و بسته، ساختار دستورات، تعریف متغیرها، انواع داده (رشته، عدد، بولین، آرایه)، عملگرها و نمایش خروجی.
  3. ساختارهای شرطی (۳۵ دقیقه): if، else، elseif، ترکیب با عملگرهای منطقی (AND، OR، NOT) و عملگرهای مقایسه‌ای.
  4. تمرین عملی سیستم ارزیابی نمرات (۳۵ دقیقه): پیاده‌سازی کامل با خروجی شرطی، مدیریت خطاها و نمایش پیام‌های مناسب.
📌 نکته مهم برای مدرس: در این جلسه، دانشجویان برای اولین بار با محیط سرور مواجه می‌شوند. حتماً تأکید کنید که فایل‌های PHP باید از طریق localhost اجرا شوند و نه با دوبار کلیک روی فایل! تفاوت بین http://localhost/... و file:///... را به‌خوبی توضیح دهید.

⚠️ عیب‌یابی رایج:
  • خطای «Port 80 is busy»: اگر پورت ۸۰ توسط نرم‌افزار دیگری (مثل Skype یا IIS) استفاده می‌شود، در Control Panel روی «Config» کلیک کنید و پورت را به ۸۰۸۰ تغییر دهید. سپس با آدرس http://localhost:8080 تست کنید.
  • خطای «Access Denied»: مطمئن شوید که به‌عنوان Administrator اجرا می‌کنید.
  • فایل PHP دانلود می‌شود: یعنی Apache به‌درستی اجرا نشده است. سرویس Apache را ری‌استارت کنید.

بخش سوم: آشنایی با سینتکس PHP و متغیرها (۳۰ دقیقه)

🗣 کلمه به کلمه چه بگویید: «حالا که محیط کار آماده است، بیایید با زبان PHP آشنا شویم. PHP یک زبان سمت سرور است؛ یعنی کدها روی سرور اجرا می‌شوند و نتیجه نهایی (معمولاً HTML) به مرورگر فرستاده می‌شود. به همین دلیل، کاربر نمی‌تواند کدهای PHP را در مرورگر ببیند - فقط خروجی آن را مشاهده می‌کند. چند قانون طلایی برای شروع: ۱. کدهای PHP داخل تگ‌های <?php و ?> نوشته می‌شوند. ۲. هر دستور باید با ; (نقطه‌ویرگول) تمام شود. ۳. متغیرها با علامت $ شروع می‌شوند. ۴. PHP به بزرگی و کوچکی حروف حساس است (Case-Sensitive). یعنی $name با $Name فرق دارد.»

۱. ساختار پایه یک فایل PHP:

<!DOCTYPE html>
<html>
<head>
    <title>اولین صفحه PHP</title>
</head>
<body>

    <h1>سلام دنیا!</h1>

    <?php
        // این یک کامنت است - کامنت‌ها توسط PHP اجرا نمی‌شوند
        echo "این متن توسط PHP تولید شده است!";
    ?>

    <p>این بخش HTML معمولی است.</p>

</body>
</html>
📌 نکته: فایل‌های PHP می‌توانند شامل کدهای HTML و PHP با هم باشند. PHP فقط بخش‌های داخل <?php ... ?> را پردازش می‌کند و بقیه را به‌عنوان HTML معمولی به مرورگر می‌فرستد.

۲. متغیرها در PHP:

متغیرها برای ذخیره موقت داده‌ها در حافظه استفاده می‌شوند. در PHP، متغیرها با $ شروع می‌شوند و نیازی به تعیین نوع داده ندارند (زبان ضعیف‌نوع یا Loosely Typed). 

<?php
// ===== انواع داده در PHP =====

// ۱. رشته (String) - با "" یا ''
$name = "علی رضایی";
$city = 'تهران';

// ۲. عدد صحیح (Integer)
$age = 25;
$year = 2024;

// ۳. عدد اعشاری (Float)
$price = 19.99;
$temperature = -5.5;

// ۴. بولین (Boolean) - true یا false
$is_logged_in = true;
$is_admin = false;

// ۵. آرایه (Array) - برای ذخیره چند مقدار
$colors = array("قرمز", "آبی", "سبز");
$numbers = [1, 2, 3, 4, 5];

// ۶. مقدار خالی (Null)
$nothing = null;

// ===== نمایش متغیرها با echo =====
echo "نام من " . $name . " است و " . $age . " سال دارم.<br>";
echo "قیمت این محصول: $" . $price . "<br>";

// استفاده از double quote برای نمایش مستقیم متغیر
echo "شهر من: $city <br>";

// ===== عملگرهای ریاضی =====
$a = 10;
$b = 3;

echo "جمع: " . ($a + $b) . "<br>";        // 13
echo "تفریق: " . ($a - $b) . "<br>";       // 7
echo "ضرب: " . ($a * $b) . "<br>";         // 30
echo "تقسیم: " . ($a / $b) . "<br>";       // 3.333...
echo "باقیمانده: " . ($a % $b) . "<br>";   // 1

// ===== عملگرهای الحاق (اتصال رشته‌ها) =====
$first_name = "رضا";
$last_name = "کریمی";
$full_name = $first_name . " " . $last_name;
echo "نام کامل: $full_name <br>";

// ===== تابع var_dump() برای مشاهده نوع و مقدار =====
var_dump($age);   // int(25)
var_dump($colors); // array(3) { ... }
?>
📌 تفاوت '' و "" در PHP:
  • در ' ' (single quote)، متغیرها نمایش داده نمی‌شوند و فقط متن خام نمایش داده می‌شود.
  • در " " (double quote)، متغیرها مقدارشان را نمایش می‌دهند.

مثال:

$name = "علی";
echo 'سلام $name';  // خروجی: سلام $name
echo "سلام $name";  // خروجی: سلام علی

۳. ثابت‌ها (Constants):

ثابت‌ها متغیرهایی هستند که بعد از تعریف، قابل تغییر نیستند. برای تعریف ثابت از define() استفاده می‌شود. 

<?php
// تعریف ثابت
define("SITE_NAME", "آموزشگاه برنامه‌نویسی");
define("YEAR", 2024);
define("PI", 3.14159);

// استفاده از ثابت - بدون نیاز به $
echo "به " . SITE_NAME . " خوش آمدید. سال " . YEAR;
?>
🧪 تمرین سریع برای دانشجویان (۵ دقیقه):

یک فایل variables.php بسازید و متغیرهای زیر را تعریف کنید:

  • $student_name با مقدار نام خودتان
  • $student_age با مقدار سن شما
  • $student_major با مقدار رشته تحصیلی شما
  • $gpa با مقدار معدل شما (عدد اعشاری)

سپس با echo یک جمله کامل مانند زیر نمایش دهید:

"سلام، من [نام] هستم، [سن] سال دارم و در رشته [رشته] تحصیل می‌کنم. معدل من [معدل] است."

بخش چهارم: ساختارهای شرطی (if, else, elseif) (۳۵ دقیقه)

🗣 کلمه به کلمه چه بگویید: «برنامه‌نویسی یعنی تصمیم‌گیری. ما با شرط‌ها مشخص می‌کنیم که در چه شرایطی چه کدی اجرا شود. زندگی روزمره پر از تصمیم‌گیری است: «اگر باران ببارد، چتر برمی‌دارم. در غیر این صورت، عینک آفتابی می‌زنم.» در برنامه‌نویسی هم دقیقاً همین کار را می‌کنیم!»

۱. ساختار if ساده:

<?php
$age = 20;

if ($age >= 18) {
    echo "شما بزرگسال هستید و می‌توانید رای دهید.";
}
?>

۲. if-else:

<?php
$score = 75;

if ($score >= 60) {
    echo "تبریک! شما قبول شدید.";
} else {
    echo "متأسفیم، شما مردود شدید.";
}
?>

۳. if-elseif-else (چند شرطی):

<?php
$grade = 85;

if ($grade >= 90) {
    echo "نمره عالی! رتبه A";
} elseif ($grade >= 80) {
    echo "نمره خوب! رتبه B";
} elseif ($grade >= 70) {
    echo "نمره متوسط! رتبه C";
} elseif ($grade >= 60) {
    echo "نمره قابل قبول! رتبه D";
} else {
    echo "نیاز به تلاش بیشتر! رتبه F";
}
?>

۴. عملگرهای مقایسه‌ای:

عملگر معنی مثال
== برابر (مقدار) 5 == "5" → true
=== برابر (مقدار و نوع) 5 === "5" → false
!= نامساوی 5 != 3 → true
> بزرگتر 10 > 5 → true
< کوچکتر 3 < 7 → true
>= بزرگتر یا مساوی 5 >= 5 → true
<= کوچکتر یا مساوی 4 <= 4 → true

۵. عملگرهای منطقی:

عملگر معنی مثال
&& و (AND) true && true → true
|| یا (OR) true || false → true
! نقیض (NOT) !true → false

۶. مثال ترکیبی با عملگرهای منطقی:

<?php
$age = 22;
$has_license = true;
$is_vip = false;

// کاربر باید بالای ۱۸ سال باشد و گواهینامه داشته باشد
if ($age >= 18 && $has_license) {
    echo "شما می‌توانید رانندگی کنید.";
}

// کاربر یا VIP است یا سنش بالای ۶۵ است
if ($is_vip || $age >= 65) {
    echo "شما مشمول تخفیف ویژه می‌شوید.";
}

// کاربر نباید زیر ۱۸ سال باشد
if (!($age < 18)) {
    echo "شما بزرگسال هستید.";
}
?>
📌 عملگر سه‌گانه (Ternary Operator):

یک روش کوتاه‌نویس برای if-else ساده:

$status = ($score >= 60) ? "قبول" : "مردود";
// معادل این کد:
if ($score >= 60) {
    $status = "قبول";
} else {
    $status = "مردود";
}

بخش پنجم: تمرین عملی – سیستم ارزیابی نمرات (۳۵ دقیقه)

🗣 مأموریت کلاسی: «یک فایل PHP به نام grade.php بسازید. این فایل یک سیستم کامل ارزیابی نمرات باشد. ویژگی‌های سیستم: ۱. یک متغیر $score با مقدار دلخواه (مثلاً ۷۵) تعریف کنید. ۲. با استفاده از شرط‌ها، پیام مناسب را نمایش دهید. ۳. تمام حالت‌های ممکن را پوشش دهید. جدول نمرات: ┌────────────┬─────────────────────────────────────┐ │ محدوده نمره │ پیام نمایشی │ ├────────────┼─────────────────────────────────────┤ │ ۱۰۰ - ۹۰ │ عالی، شما قبول شدید با رتبه A │ │ ۸۹ - ۷۰ │ خوب، شما قبول شدید با رتبه B │ │ ۶۹ - ۵۰ │ قابل قبول، شما قبول شدید با رتبه C │ │ ۴۹ - ۰ │ متأسفیم، شما مردود شدید │ │ خارج از ۰-۱۰۰ │ نمره نامعتبر است! │ └────────────┴─────────────────────────────────────┘ همچنین یک پیام اضافی نمایش دهید: - اگر نمره ۱۰۰ بود: "نمره کامل! تبریک ویژه!" - اگر نمره کمتر از ۲۰ بود: "نیاز به تلاش جدی دارید!" - اگر نمره بین ۴۰ تا ۴۹ بود: "تقریباً قبول شدید، یک قدم تا موفقیت فاصله دارید!"»

  4. مقدار $score را تغییر دهید و نتایج مختلف را مشاهده کنید:

    • $score = 15; → پیام ویژه

سیستم را طوری تغییر دهید که نمره را از کاربر دریافت کند (با فرم HTML).

راهنما:

  • یک فرم ساده با یک فیلد ورودی و دکمه ارسال بسازید.
  • از متد GET استفاده کنید و نمره را با $_GET['score'] دریافت کنید.
  • اگر نمره ارسال نشد، پیام "لطفاً نمره خود را وارد کنید" نمایش دهید.
